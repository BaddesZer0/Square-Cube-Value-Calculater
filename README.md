Program Hakkında:

Bu program kullanıcıdan alınan kenar uzunluğu ile oluşan kare ve küp için aşağıda belirtilen çeşitli değerleri hesaplamaktadır:

Program tarafından hesaplanabilen değerler:

a)Karenin çevresi

b)Karenin alanı

c)Küpün dış yüzey alanı

d)Küpün hacmi

e)Küpün çevresi

Nasıl Çalıştırılır?

1)Hangi kenar uzunluğu için değerlerin hesaplanacağını giriniz.

2)Program kenar uzunluğunu pozitif bir sayı olarak girmemeniz durumunda sizi uyaracaktır.

3)Programı çalıştırmak için ilk iki adıma uygun haraket etmeniz yeterlidir.
